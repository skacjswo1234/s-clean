# Google Sheets 헤더 설정 가이드

## 📋 필요한 헤더 (첫 번째 행에 입력)

Google Sheets의 **첫 번째 행(A1~F1)**에 다음 헤더를 입력하세요:

| A1 | B1 | C1 | D1 | E1 | F1 |
|---|---|---|---|---|---|
| **접수일시** | **이름** | **연락처** | **서비스 유형** | **평수/면적** | **문의내용** |

## 📝 수동 입력 방법

### 방법 1: 직접 입력 (가장 간단)

1. Google Sheets를 엽니다.
2. 첫 번째 행에 다음 순서대로 입력합니다:

```
A1: 접수일시
B1: 이름
C1: 연락처
D1: 서비스 유형
E1: 평수/면적
F1: 문의내용
```

### 방법 2: 복사해서 붙여넣기

다음 텍스트를 복사하여 A1 셀에 붙여넣으면 자동으로 6개 컬럼으로 분리됩니다:

```
접수일시	이름	연락처	서비스 유형	평수/면적	문의내용
```

## 🎨 헤더 스타일링 (선택사항)

헤더를 더 보기 좋게 만들고 싶다면:

1. **1행 전체 선택**: 행 번호 1을 클릭하여 첫 번째 행 전체를 선택합니다.
2. **굵게**: Ctrl+B (또는 서식 > 굵게)
3. **배경색**: 서식 > 셀 색상 > `#cfbeb0` (베이지색)
4. **글자색**: 서식 > 텍스트 색상 > 흰색
5. **열 너비 자동 조정**: 더블클릭으로 자동 조정

## 🔧 자동 설정 방법 (Apps Script 사용)

헤더를 자동으로 만들고 싶다면:

1. Apps Script 에디터를 엽니다 (확장 프로그램 > Apps Script)
2. `setupSheet` 함수를 선택합니다
3. 상단의 **실행** 버튼을 클릭합니다
4. 권한을 허용하면 자동으로 헤더가 생성됩니다

## 📊 최종 결과 예시

헤더를 설정하면 다음과 같이 보입니다:

```
┌────────────────────┬──────────┬──────────────┬──────────────┬───────────┬────────────┐
│ 접수일시           │ 이름     │ 연락처       │ 서비스 유형  │ 평수/면적 │ 문의내용   │
├────────────────────┼──────────┼──────────────┼──────────────┼───────────┼────────────┤
│                    │          │              │              │           │            │
│ (여기에 문의 데이터가 자동으로 추가됩니다)                                    │
└────────────────────┴──────────┴──────────────┴──────────────┴───────────┴────────────┘
```

## ✅ 확인 사항

헤더가 제대로 설정되었는지 확인:

- ✅ A1 = "접수일시"
- ✅ B1 = "이름"
- ✅ C1 = "연락처"
- ✅ D1 = "서비스 유형"
- ✅ E1 = "평수/면적"
- ✅ F1 = "문의내용"

## ⚠️ 주의사항

1. **첫 번째 행에만** 헤더를 입력하세요 (두 번째 행부터는 데이터가 들어갑니다)
2. 헤더 순서는 반드시 위와 같아야 합니다 (코드에서 이 순서로 데이터를 저장합니다)
3. 헤더 이름은 정확히 일치해야 합니다 (대소문자 구분 없음)

## 🚀 헤더 설정 후 다음 단계

헤더를 설정한 후에는:
1. Apps Script 코드에 스프레드시트 ID 입력
2. 웹 앱으로 배포
3. HTML 파일에 웹 앱 URL 입력

자세한 내용은 `GOOGLE_SHEETS_SETUP.md` 파일을 참고하세요.
